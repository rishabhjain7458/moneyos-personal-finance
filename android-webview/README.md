# MoneyOS Android WebView

Open this `android-webview` folder in Android Studio after the web app is deployed.

1. Deploy the web app with Firebase Hosting.
2. Replace `https://your-project.web.app` in `app/src/main/res/values/strings.xml`.
3. In Android Studio, use Build > Build Bundle(s) / APK(s) > Build APK(s).

Google login in Android WebView needs your Firebase Auth authorized domains to include the Firebase Hosting domain you load here.
